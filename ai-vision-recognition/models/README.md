# MobileNet-SSD model files

Place these legitimate pre-trained Caffe MobileNet-SSD files in this folder:

- `MobileNetSSD_deploy.prototxt`
- `MobileNetSSD_deploy.caffemodel`

They are intentionally not included in this repository. Download them from a trusted source, such as the original MobileNet-SSD project or a reputable OpenCV model collection. The application checks for both exact filenames before loading object detection.
